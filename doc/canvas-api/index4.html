Canvas API
ImageBitmap
 

    了解ImageBitmap

    简介

    会返回：，例如我们在Chrome浏览器控制台执行window.ImageBitmap，ImageBitmap表示一个位图图像，可以将其绘制到Canvas而不会产生不适当的延迟。 可以使用createImageBitmap()工厂方法从各种源对象创建它。ImageBitmap提供了一种异步且高性能的途径来预渲染WebGL中纹理。这是一个在浏览器可视窗体中可以全局访问的接口

    ƒ ImageBitmap() { [native code] }

    属性

    ImageBitmap有如下符合规范标准的属性。

    ImageBitmap.height

    只读。是无符号长整数，表示ImageBitmap对象的CSS像素高度。

    ImageBitmap.width

    只读。是无符号长整数，表示ImageBitmap对象的CSS像素宽度。

第19章 表单脚本

Web 表单在HTML 中以

元素表示，在 JS 中则以 HTMLFormElement 类型表示，它除了拥有其他 HTML 元素一样的默认属性外，还有自己的属性和方法：

  • acceptCharset：服务器可以接收的字符集
  • action：请求的 URL
  • elements：表单中所有控件的 HTMLCollection
  • enctype：请求的编码类型
  • length：表单中控件的数量
  • method：HTTP 请求的方法类型，通常是"get"或"post"
  • name：表单的名字
  • reset()：把表单字段重置为各自的默认值
  • submit()：提交表单
  • target：用于发送请求和接收响应的窗口的名字

表单是通过用户点击提交按钮或图片按钮的方式提交的。提交按钮可以使用 type 属性为 "submit" 的或
